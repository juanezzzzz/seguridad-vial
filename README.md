# Seguridad Vial

Proyecto web sobre seguridad y movilidad vial. Sitio estático hecho con HTML, CSS y JavaScript puro (sin frameworks ni build).

## Estructura

```
seguridad-vial/
├── index.html        # Página principal
├── css/
│   └── styles.css    # Estilos
├── js/
│   └── app.js        # Lógica y datos (normas, consejos, emergencias)
├── assets/           # Imágenes y recursos
└── README.md
```

## Cómo ejecutarlo

Abre `index.html` en el navegador, o usa la extensión **Live Server** de VS Code
(clic derecho sobre `index.html` → *Open with Live Server*).

## Contenido actual

- **Normas** clave de tránsito
- **Consejos** prácticos de conducción
- **Números de emergencia**

El contenido se genera dinámicamente desde arreglos en `js/app.js`, así que para
agregar o editar tarjetas basta con modificar esos datos.
